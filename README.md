This program will allow the user to inspect and interact with 3D models via the program window.  

Build instructions (Linux):
1. Create a new directory in the project's root folder:
    $ mkdir build

2. Change working directory to the build directory:
    $ cd build/

3. Init Cmake:
    $ cmake ..

4. Change working directory back to project root folder:
    $ cd ..

5. Execute the "run" script
    $ ./run.sh

NOTES: 
>The run script also executes clang-format on the source/header files
>You'll need to make sure you have the libraries installed on your system
